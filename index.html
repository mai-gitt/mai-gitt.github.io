AdBlocker

I made this simple adblocker using JavaScript. The adblocker is a chrome extension and works by blocking requests sent to the sites listed on the left.

It is very simple, as it can only block ads from the list of sites. Therefore, it could better function as a parental control on children's web activities, by blocking a list of sites determined by the parents.

When a user is trying to access a site which is blocked by the extension, they are greeted with the error message on the right.
